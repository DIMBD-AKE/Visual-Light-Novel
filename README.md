개요
------
<a href="www.renpy.org">Ren'Py</a> 와 같은 훌륭한 엔진이 있지만, 이 엔진의 개발 목적은 코딩 없는 누구나 사용 가능한 엔진이다.<br><br>
이름이 Visual Light Novel 인 이유는 기존 Visual Novel 들의 단편적인 구조에서<br>
책을 모으듯 여러개의 이야기를 관리할 수 있게 하는것에 있다. (DLC와 유사한 기능)<br><br>
현재까지의 계획으로는 WPF을 이용한 데이터 제작과<br>
DirectX 11을 이용해 블루프린트를 작성하여 UI, 움직임을 구현하는것이다.<br><br>
